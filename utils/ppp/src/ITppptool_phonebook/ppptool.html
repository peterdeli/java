ITppptool for Solaris and Linux

Description

PPPtool is a graphical interface for connecting to
remote networks via dialup modem.

The Main window consists of a Menu, a display of active account information,
the 'Connect/Disconnect' Button, some mock 'status lights', and a 'status' line.

Menu Items The Menu bar contains four menus:

1. 'File'
2. 'Edit',
3. 'Accounts'
4. 'Help'.

The 'File' menu contains three items:

1. Save Accounts
2. View Log
3. Exit

The 'Edit' menu contains two items:

1. Manage Accounts
2. Edit Preferences

The 'Accounts' menu is a dynamic menu and holds the names of all accounts which have been created.

The 'Help' menu contains two items:
1. 'About PPP Tool'
2. 'PPP Tool Help'

ITppptool Configuration

I. Creating Accounts

To create a new account, Select 'Edit/Manage Accounts' Click on 'New'.

A window will appear with the following fields: ( ** denotes a required field )

[ ** Account Name ]: Enter the name of the account
[ ** User ID ]: Enter the login name or ID of the account
[ Password ]: Enter the password of the account ( if one exists )
[ ** Phone Number ]: Enter the Phone Number of the account
[ DNS Domain Name ]: Enter the domain name of the account ( if one exists )
[ DNS Search Domains ]: Enter any domains which DNS should search
[ Account Nameserver #1 ]: Enter a primary nameserver for the account
[ Account Nameserver #2 ]: Enter a secondary nameserver for the account Near the bottom of the window are the following checkbuttons:
[ ** Authentication Type ]: Select the authentication protocol or method
[ Default Route ]: If selected, adds a 'default route' option to pppd
[ Create /etc/resolv.conf ]: If selected, copies the DNS information to /etc/resolv.conf
 
 

II. Saving Accounts

To save the current account information to the account file, select File/Save Accounts.
If there have been any changes to the accounts, a prompt will pop up to confirm the save.
Clicking 'Yes' will save the account data, clicking 'Cancel' will not save the account data and return to PPP tool.

III. Editing Accounts

To make changes to an account, first select Edit/Manage Accounts.
Click on an account name in the window, then click on 'Edit'
A window will appear with the account information.
Edit any fields or select/deselect any items, then click on 'Save Edits'
The window will close and the changes will be saved in the currently running program, but will NOT be save to the accounts file.
To save the changes to the accounts file, select 'File/Save Accounts' and click on the 'OK' button .

IV. Deleting Accounts

To delete and account, first select Edit/Manage Accounts.
Click on an account name in the window, then click on 'Delete'
The account will be deleted in the currently running program, but will NOT  be saved to the accounts file.
To save the changes to the accounts file, select 'File/Save Accounts' and click on the 'OK' button .

V. Making an  Account Active

To make an account active means to make it available for connecting.

There are two ways to do this:

1. Select the account from the 'Accounts' menu item in the main window.
2. In the 'Account Manager' window, select the account and click the 'Make Active' button.

VI. Preferences

Upon first starting ppptool, a set of default preferences will be loaded.
To make sure the correct settings are in effect, check the preferences window by selecting 'Edit/Edit Preferences'.

The settings in the preferences are:

Modem Port: (required): This is the port to which the modem is connected.
Flow Control: hardware, software, or none.
Port Speed (required)
Init String: Enter any modem initialization string here.
Connect String: Reserved for future use
PPP options: Reserved for future use

Connecting with PPPtool

To initiate a  ppp connection, follow these steps:

1. Select an account and make it active ( See V. above )
2. Make sure your modem is connected to the correct port, and that the correct cable is connected between the modem and the computer.
3. Verify the account information is correct, then click on the 'Connect' Button.
4. If possible, turn up the volume on the modem so that you can hear it going off-hook, dialing, and connecting.
5. Monitor the 'Status' line at the bottom of the main window, and the log window as well ( File/View Log ).
6. When you are successfully connected, the status line will expand with information about the connection, such as:
    The IP address assigned to your host
    The name of the ppp interface
 
 


Exiting PPPtool

To exit PPPtool, select 'File/Exit'.

If there is any account information which has not been save to the account file,
a dialog will pop up asking whether to save, cancel, or return to PPPtool.

Clicking 'Yes' will save the account settings to the file.

Clicking 'No' will exit PPPtool without saving the account changes, and clicking 'Cancel' will return to PPPtool
without saving any changes to the accounts file.
